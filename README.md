# QR Code Generator

#### Легкое и отзывчивое веб-приложение для мгновенного создания QR-кодов из текста или ссылок. Проект создан для демонстрации навыков работы с DOM, API и современной версткой.

# Стек технологий

#### HTML5 – семантическая разметка.
#### CSS3 – Flexbox, адаптивность, кастомные UI-элементы.
#### JavaScript (ES6) – логика приложения и взаимодействие с API.
#### QR Server API – внешний сервис для генерации изображений.

# Основной функционал
#### Мгновенный результат: Генерация кода сразу по нажатию кнопки.
#### Валидация: Проверка на пустую строку перед отправкой запроса.
#### UX/UI: Интуитивно понятный интерфейс с анимациями при наведении и фокусе.

# Структура проекта
#### index.html — разметка интерфейса и подключение ресурсов
#### style.css — стилизация приложения, включая Flexbox-сетку и адаптивность
#### script.js — логика обработки ввода и взаимодействие с QR API
